# Controlling who can see the Vynix widget

## Question
How do I limit who sees the widget?

## Answer
This guide explains controlling who can see the vynix widget in Vynix.

Drop a lightweight widget on any site, click on what is wrong, and Vynix captures the element, a screenshot, the console and network context, and an AI diagnosis of the likely root cause. From there you can copy a ready-to-build prompt or open a GitHub issue and assign it to a coding agent.
