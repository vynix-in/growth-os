# How to install the Vynix widget on any website

## Question
How do I add Vynix to my site?

## Answer
This guide explains how to install the vynix widget on any website in Vynix.

Vynix is a website annotation and developer-context tool. Vynix is a website annotation and developer-context tool. Drop a lightweight widget on any site, click on what is wrong, and Vynix captures the element, a screenshot, the console and network context, and an AI diagnosis of the likely root cause. From there you can copy a ready-to-build prompt or open a GitHub issue and assign it to a coding agent.

For full setup steps, see the [Vynix documentation](https://vynix.in/docs). If you are still stuck, contact the team at hello@vynix.in.
